# 🎮 Tic Tac Toe – Ocean Glassmorphism

A visually enhanced Tic Tac Toe game built with vanilla HTML, CSS, and JavaScript. Designed with an Ocean Glassmorphism theme featuring animated backgrounds, glowing UI elements, and smooth micro-interactions.

## ✨ Features
- 🌊 Ocean Glassmorphism UI with frosted glass panels and floating bubble animations
- 🏆 Score Tracker — persists O, X, and Draw counts across rounds
- 🤝 Draw Detection — correctly identifies when the board is full with no winner
- 🔄 Turn Indicator — always shows whose turn it is with a glowing badge
- 🔊 Sound Effects — distinct audio feedback for moves, wins, and draws (Web Audio API)
- ✨ Animations — pop-in marks, ripple on click, winning cell pulse glow

## 🛠️ Tech Stack
- **HTML5** — Semantic structure
- **CSS3** — Custom properties, backdrop-filter, keyframe animations, clamp()
- **JavaScript (ES6+)** — DOM manipulation, Web Audio API, event handling
- **Google Fonts** — Orbitron + Nunito

## 🚀 Getting Started
Clone the repo and open `index.html` in your browser — no build tools or dependencies required.

```bash
git clone https://github.com/YOUR_USERNAME/tic-tac-toe-ocean.git
cd tic-tac-toe-ocean
open index.html
```
